# Console Task Manager

A simple console application for managing tasks, created as a learning project while studying Java.

## Features

* Add new tasks
* Display all tasks
* Mark tasks as completed
* Delete tasks
* Display only unfinished tasks
* Input validation using exceptions

## Technologies

* Java 25
* ArrayList
* Object-Oriented Programming (OOP)
* Exception Handling
* Git & GitHub

## Project Structure

```
src/
├── Main.java          // Application entry point
├── Task.java          // Task model
└── TaskManager.java   // Business logic
```
